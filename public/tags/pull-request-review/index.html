I Am the First Reviewer of My Own Pull Requests

When I completed my first task, I nervously sent my pull request to my manager. Before even opening it, he asked, “Did you review it yourself?” I was a bit surprised and said no. He told me to go review my own code first, fix any issues that you found, and then send it back to me.
